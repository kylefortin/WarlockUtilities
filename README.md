# WarlockUtilities

*Current version is : `3.2.0`*  

## Simple tools for Warlocks.  
### [Shard Manager](https://github.com/kylefortin/WarlockUtilities/blob/3.2.0/ReadMe/ShardManager.md) - clear out excess shards from bags quickly and efficiently.  
![Warlock Utilities - Shard Manager](https://github.com/kylefortin/WarlockUtilities/blob/3.2.0/Images/KeepShardsByNumber.jpg?raw=true)  

### [Stone Manager](https://github.com/kylefortin/WarlockUtilities/blob/3.2.0/ReadMe/StoneManager.md) - create and distrubute Healthstones and Soulstones.  
![Warlock Utilities - Stone Manager](https://github.com/kylefortin/WarlockUtilities/blob/3.2.0/Images/StoneManager.jpg?raw=true)  

### [Demon Manager](https://github.com/kylefortin/WarlockUtilities/blob/3.2.0/ReadMe/DemonManager.md) - help with summoning (or sacrificing) minions.
![Warlock Utilities - Demon Manager](https://github.com/kylefortin/WarlockUtilities/blob/3.2.0/Images/DemonManager.jpg?raw=true)  

### [AppTray](https://github.com/kylefortin/WarlockUtilities/blob/3.2.0/ReadMe/AppTray.md) - a quick and easy menu to launch tools from.  
![Warlock Utilities - App Menu](https://github.com/kylefortin/WarlockUtilities/blob/3.2.0/Images/AppTray.jpg?raw=true)  

### [StatsPanel](https://github.com/kylefortin/WarlockUtilities/blob/3.2.0/ReadMe/StatsPanel.md) - a simple frame to show usage statistics on Shard consumption.
![Warlock Utilities - Stats Panel](https://github.com/kylefortin/WarlockUtilities/blob/3.2.0/Images/StatsPanel.png?raw=true)  

### [SpellAnnouncer](https://github.com/kylefortin/WarlockUtilities/blob/3.2.0/ReadMe/SpellAnnouncer.md) - let your friends know what you're up to!
